# Changelog Component

Projeto desenvolvido para praticar estruturação de layouts utilizando apenas HTML e CSS.

## Objetivo

Criar um componente de changelog responsivo, exibindo uma linha do tempo com atualizações de um projeto ou software.

## O que foi praticado

- Estrutura semântica em HTML
- Posicionamento com CSS
- Flexbox
- Pseudo-elementos (`::before`)
- Organização de layout
- Timeline vertical
- Centralização e espaçamento
