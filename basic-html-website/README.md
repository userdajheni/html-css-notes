O objetivo deste projeto é ensinar-lhe como estruturar um site usando HTML, ou seja, diferentes seções de um site como cabeçalho, rodapé, navegação, conteúdo principal, barras laterais etc.

Neste projeto, você é obrigado a criar um site simples somente HTML com várias páginas. O site deve ter as seguintes páginas:
    Página inicial
    Projetos
    Artigos
    Contato

O site deve ter uma barra de navegação que deve estar presente em todas as páginas e link para todas as páginas.
