# Photo Showcase

Projeto desenvolvido para praticar o uso de imagens, legendas e vídeos utilizando HTML semântico.

## Objetivo

Criar uma página de galeria de fotos utilizando boas práticas de acessibilidade e estruturação de mídia.

## Como executar

1. Clone este repositório
2. Abra o arquivo `index.html` no navegador
